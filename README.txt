UPDATE 01
Grid Turn-Based Game (C++)

ผู้พัฒนา: ณัฐภัทร ขำดี 680610747
วิชา: Computer Programming
โปรแกรมภาษา: C++
คอมไพเลอร์: g++

📌 รายละเอียดโปรแกรม
โปรแกรมนี้เป็นเกมแบบ Turn-Based บนแผนที่ 2 มิติ ผู้เล่น (P) ต้องต่อสู้กับศัตรู (E) และผ่านด่านต่อไปเรื่อยๆ

🎮 วิธีเล่น

P = Player
E = Enemy

# = กำแพง

. = พื้นที่เดินได้

การควบคุม:
W = ขึ้น
S = ลง
A = ซ้าย
D = ขวา

กติกา:

ผู้เล่นและศัตรูจะสลับเทิร์นกันเดิน

ถ้าศัตรูเดินมาชนผู้เล่น → การทำDamageใส่ศัตรู ตามจำนวนDamage ที่ player มี จนกว่า player หรือ Enemy ตาย


⚙ วิธีคอมไพล์และรัน

เปิด Command Prompt ที่โฟลเดอร์โปรเจ็ก แล้วพิมพ์:

g++ main.cpp Game.cpp Map.cpp Player.cpp Enemy.cpp Entity.cpp -o game.exe

จากนั้นรันเกม:

game.exe

 โครงสร้างไฟล์

main.cpp → เริ่มโปรแกรม
Game.cpp / Game.h → ควบคุมเกมหลัก
Map.cpp / Map.h → แผนที่และการแสดงผล
Player.cpp / Player.h → การเคลื่อนที่ผู้เล่น
Enemy.cpp / Enemy.h → AI ศัตรู
Entity.cpp / Entity.h → คลาสพื้นฐานของตัวละคร

 แนวคิดการทำงาน

ใช้ OOP แยกหน้าที่ของแต่ละคลาส

ใช้ระบบเทิร์น Player → Enemy

ตรวจสอบการชนและเงื่อนไขชนะ/แพ้ใน Game class

แผนที่ถูกเก็บใน array 2 มิติ
